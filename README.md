# jellyfin-badge-New-kids
Add a new badge to newly added films that disappears after 7 days, and also add a Kids badge to films or cartoons with a Kids tag already inserted in the tags

<h2>Install Instructions:</h2>

1. Install the JS Injector plugin - https://github.com/n00bcodr/Jellyfin-JavaScript-Injector

2. Within Plugins > JS Injector, add a script, name it "Badge-New-kids" or similar for reference, then paste the raw code of the jellyfin-badge-New-kids.js file in this project, then save.

3. Hard refresh your browser (ctrl + f5, shift + refresh, or clear site data) and test!

4. Info: If the Kids badge doesn't appear, manually add a new "Kids" tag
